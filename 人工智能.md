## Keras

序贯模型是多个网络层的线性堆叠,步骤:

1. model.add，添加层；
2. model.compile,模型训练的BP模式设置；
3. model.fit，模型训练参数设置 + 训练；
4. 模型评估
5. 模型预测

添加层

```python
model = Sequential([
    Dense(32, units=784),
    Activation('relu'),
    Dense(10),
    Activation('softmax'),
])
//等同于
model = Sequential()
model.add(Dense(32, input_shape=(784,))) //input_shape是tuple类型
model.add(Activation('relu'))
```
